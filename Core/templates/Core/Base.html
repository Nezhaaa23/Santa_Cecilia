Introducción

Bienvenido al Sistema de Inventario. Esta herramienta está diseñada para facilitar la gestión de productos, reportes y control de stock en tiempo real. A continuación, se explica cómo utilizar cada una de las funcionalidades del sistema de manera eficiente.

Gestión de Inventario

En esta sección puedes agregar, editar y eliminar productos del inventario. Sigue las instrucciones:

  • Accede al menú principal y selecciona la opción "Inventario".
  • Para agregar un producto, haz clic en el botón "Agregar Producto".
  • Rellena los campos requeridos:
    • Nombre: Identificación del producto.
    • Cantidad: Número de unidades disponibles.
  • Para editar o eliminar un producto, utiliza los botones correspondientes junto a cada ítem.
